# BasicProgramsBeforeDSA

This project contains simple Java programs for beginners, including basic input/output, arithmetic, and number swapping examples.

## Project Structure

- src/Basics/ - Java source files
- bin/Basics/ - compiled class files

## How to Run

Compile a Java file:

```bash
javac -d bin src/Basics/HelloWorld.java
```

Run the compiled program:

```bash
java -cp bin HelloWorld
```

You can replace `HelloWorld` with the name of any other class such as `AddTwoNumbers`, `SwapTwoNumbers`, or `UserInput`.

## Notes

These programs are intended as beginner practice and are organized in the Basics folder.
